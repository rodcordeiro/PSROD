# Open

## SYNOPSIS

## SYNTAX

```
Open [[-Path] <String>] [-AsFolder] [<CommonParameters>]
```

## DESCRIPTION
Open the specified path in editor or file explorer

## EXAMPLES

### EXEMPLO 1
```
Open ./desktop
```

### EXEMPLO 2
```
Open Project/Project.sln
```

### EXEMPLO 3
```
Open archive.json
```

### EXEMPLO 4
```
Open Project -AsFolder
```
