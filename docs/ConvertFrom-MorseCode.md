# ConvertFrom-MorseCode

## SYNOPSIS
Writes text from Morse code

## SYNTAX

```
ConvertFrom-MorseCode [-MorseCode] <String> [<CommonParameters>]
```

## DESCRIPTION
This PowerShell script writes text from Morse code.
